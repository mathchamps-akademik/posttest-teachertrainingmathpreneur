# Post Test

Folder ini dibuat standalone untuk GitHub Pages.

Cara upload:
1. Buka repository GitHub Pages yang sesuai.
2. Upload isi folder ini ke root repository.
3. Pastikan file bernama index.html berada langsung di root repository.
4. Tunggu GitHub Pages selesai deploy, lalu buka link repository.

Catatan:
- Halaman test sudah self-contained. Tidak ada dependency JS/CSS eksternal selain Google Fonts.
- Tombol Mulai Test tidak bergantung pada file assets/test-runner.js.
- Hasil submit tersimpan ke storage yang sama dengan dashboard admin.
- Jika pengajar kelas online, isi kolom Cabang dengan ONLINE.
